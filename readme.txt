== Description ==

Add attributes to some block to hidden on Desktop, Tablet or mobile. Also have the attribute to hide blocks when the user is loggend in or not.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/slothtenberg-extensions` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. If you use the theme Tertius by SlothGeek the plugin install automaticaly.
